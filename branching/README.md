# Выполнение задания "Ветвление, merge и rebase"

## Описание работы
1. Созданы файлы merge.sh и rebase.sh с начальным содержимым
2. Создана ветка git-merge с двумя коммитами, исправляющими скрипт
3. В main внесено изменение в rebase.sh
4. Создана ветка git-rebase от старого коммита с двумя коммитами
5. Ветка git-merge слита в main через merge
6. Ветка git-rebase перебазирована на main с разрешением конфликтов и объединением коммитов
7. После force push ветка git-rebase слита в main через fast-forward
